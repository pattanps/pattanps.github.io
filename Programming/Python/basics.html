# ตัวดำเนินการ ทางคณิตศาตร์พื้นฐาน

ในขั้นต้น จะให้นักเรียนลองทำความรู้จักกับ คำสั่งตัวดำเนินารต่อไปนี้

• +: plus 
• -: minus
• /: slash
• *: asterisk
• %: percent
• <: less-than
• >: greater-than
• <=: less-than-equal
• >=: greater-than-equal

โดยศึกษา code ต่อไปนี้

print("ฉันจะนับหมากับแมว:")

print("หมา", 25 + 7 / 6)

print("แมว", 100 - 25 * 3 % 4)

print("ต่อไปจะนับ หนู:")

print(3 + 2 + 1 - 5 + 4 % 2 - 1 / 4 + 6)

print("จริงไหมว่า 3 + 2 < 5 - 7?")

print(3 + 2 < 5 - 7)

print("3 + 2 ได้เท่าไหร่", 3 + 2)

print("5 - 7 ได้เท่าไหร่", 5 - 7)

print("อ่าว แล้ว 'False' คืออะไร")

# ชื่อและตัวแปร

ในการทำ program ตัวแปร(variable)คือชื่อของบางสิ่งที่กำหนดมา คนพัฒนาหรือเขียนโปรแกรมจะเขียนโค้ดเพื่อแทนสิ่งของต่างๆด้วยภาษาอังกฤษ(ภาษาที่คนอ่านเข้าใจ)เพื่อช่วยเหลือให้พวกเขาจำ และเรียกมาใช้งานสืบไป

ตัวอย่าง

cars = 100
space_in_a_car = 4.0
drivers = 30
passengers = 90
cars_not_driven = cars - drivers
cars_driven = drivers
carpool_capacity = cars_driven * space_in_a_car
average_passengers_per_car = passengers / cars_driven
print("มีรถจำนวน", cars, "คัน ที่ ใช้ได้")
print("มีจำนวนคนขับรถได้", drivers, " คน ที่ใช้ได้")
print("ดังนั้นจะมีรถที่ว่าง", cars_not_driven, "คัน วันนี้")
print("และเราสามารถขนส่งคนได้จำนวน", carpool_capacity, "คน วันนี้")
print("มีผู้โดยสาร", passengers, "คน จะมาที่มาหารถเช่าวันนี้")
print("โดยเฉลี่ยแล้ว วันนี้จะ มีผูโดยสาร", average_passengers_per_car,"คน ต่อรถแต่ละคัน")

เมื่อรันให้ผลออกมาจะได้

>>
มีรถจำนวน 100 คัน ที่ ใช้ได้
มีจำนวนคนขับรถได้ 30  คน ที่ใช้ได้
ดังนั้นจะมีรถที่ว่าง 70 คัน วันนี้
และเราสามารถขนส่งคนได้จำนวน 120.0 คน วันนี้
มีผู้โดยสาร 90 คน จะมาที่มาหารถเช่าวันนี้
โดยเฉลี่ยแล้ว วันนี้จะ มีผูโดยสาร 3.0 คน ต่อรถแต่ละคัน
