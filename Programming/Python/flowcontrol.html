# Flow Control (if-elif-else)

  • ตัวอย่างและ flow diagram
  • ตัวอย่างสร้างเงื่อนไข C ในใช้งาน if-elif-else
ทำการควบคุมการทำงานของโปรแกรมด้วยเงือนไขเป็นความรู้พื้นฐานเพื่อบรรลุการทำงานที่ซับซ้อนขึ้น เดิมทีการเลือกการทำงานใช้เหตุผลดังเช่น

if(ถ้า) ฝนตก:
    เราจะไม่ออกจากบ้าน
if(ถ้า) มีร่ม and(และ) ฝนตก:
    เราออกจากบ้านได้
else:
    ออกจากบ้าน

เราสามารถสร้าง syntax เพื่อเขียนโปรแกรมได้ดังนี้

if condition :
    statement1
else :
    statement2

# ตัวอย่างสร้างเงื่อนไข C ในใช้งาน if-elif-else

if a%2 == 0 :		#เป็นเลขคู่หรือไม
if a%100 == 0 :		#a หารด้วย 100 ลงตัวหรือไม
if (a//100)%10 == 9 :	#เลขหลักร้อยของ a คือ 9 หรือไม่ หรือ
if a <= x <= b : 	#x มีค่าในช่วงตั้งแต่ a ถึง b หรือไม

mx = a
if b > mx : 
    mx = b
if c > mx : 
    mx = c
if d > mx : 
    mx = d 	
#เก็บค่ามากสุดของ a,b,c และ d หรือเขียนโดยใช้ฟังก์ชัน max	
mx = max(a,b,c,d)		
#max(a,b,c,d) หาค่ามากสุดของ a,b,c และ d
