# เริ่มการใช้งาน Python

ในชั้นเรียนของเรา จะเรียนรู้การเขียนโปรแกรมผ่าน jupyter notebook python แต่ก่อนจะไปใช้กัน อยากให้รู้จักวิธีต่างๆเพื่อเรียกใช้ python กันก่อน

# ทำความรู้จัก Terminal console

ให้นักเรียนเข้าที่ กดที่ Terminal

จากนั้น Browser จะเข้าสู้หน้าใหม่ และจอสีดำพร้อมตัวหนังสือสีขาว ดังรูป

nbuser@nbserver:~$

ตัวอักษรบ่งบอกถึง พร้อมจะรับคำสั่ง

จากนั้นให้นักเรียนลองพิมพ์คำว่า

nbuser@nbserver:~$ python

จากนั้นก็กดปุ่ม Enter ที่ keyboard และลองดูผลจะได้เป็น (ตัวอย่าง)

nbuser@nbserver:~$ python
Python 2.7.12 (default, Nov 12 2018, 14:36:49)
[GCC 5.4.0 20160609] on linux2
Type "help", "copyright", "credits" or "license" for more information.
>>>

(ให้นักเรียนลองพิมำ์คำสั่งสู่เครื่อง ตามคำแนะนำด้านบน)

# คำถาม? python ที่นักเรียนเรียกจากเครื่อง server คือเวอร์ชั่นอะไร และจะเรียกเวอร์ชั่นอื่นได้อย่างไร

# การสร้างไฟล์ .py ผ่าน terminal console

script ชุดคำสั่งภาษา python จะใช้ file ที่มีนามสกุล .py

ตัวอย่าง เราต้องการไฟล์ชื่อ myfirstprogram.py

nbuser@nbserver:~$ touch myfirstprogram.py

    print('Hello world!!')

และเมื่อต้องการ run script python เราใช้คำสั่ง

nbuser@nbserver:~$ python myfirstprogram.py

# รู้จักคำสั่ง print

เมื่อต้องการจะให้ จอแสดงผล จะใช้คำสั่ง print ตัวอย่าง

   >>> print(2+1)
       3
   >>> print("Hello")
       "Hello"
