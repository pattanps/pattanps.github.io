ในชั้นเรียนนี้จะสงเสริมให้นักเรียนมีความสามารถในการคิดแบบ นักวิทยาศาสตร์คอมพิวเตอร์ (computer scientist) โดยใช้ วิธีการคิดเชิงคำนวนเพื่อแก้ปัญหาด้านวิทยาศาตร์และวิศวกรรม

# High level programming และ Low level programming

การสื่อสารกับเครื่องจักรที่มีหน่วยประมวลผลเป็นอุปกรณ์อิเล็กทรอนิกส์ มีการกำหนดภาษาขึ้นมาใช้งาน โดยแบ่งตามลำดับการสื่อสาร ได้แก่

  1. Low level programmimg language เป็นภาษาที่ใช้เพื่อสื่อสารกับเครื่องโดยตรง หรืออาจจะเรียกว่า "machine languages" หรือ "assambly language"

  2. High level programming language เป็นภาษาที่ระดับสูงขึ้นมาเพื่อ สะดวกต่อการใช้งานมากขึ้น โดยจะมีความง่ายต่อการนำคอมพิวเตอร์มาใช้ตามต้องการ ตัวอย่างภาษาเช่นนี้ได้แก่ C, C++ และ Java โดนรวมไปถึง ภาษา Python ที่เราจะได้เรียนกันต่อไปด้วย

# การตีความภาษาคอมพิวเตอร์กับเครื่อง

ในการดำเนินการของ High-level language เพื่อส่งให้ Low-level language เพื่อสื่สารกับเครื่องให้ทำงานมีอยู่สองแบบ คือ interpreters และ complier ดังแสดงขั้นตอนการทำงานดังนี้

  1. interpreter ตัวอย่างภาษาคือ Python

  1. complier ตัวอย่างภาษาที่ใช้ คือ C, C++

สำหรับ python นั้นใช้การทำงานเป็นแบบ interpreter โดยมีสองวิธีในการใช้งาน คือ interactive mode และ script mode โดยในขั้นแรกเราจะมาใช้งานทั้งสองแบบกันก่อน แต่ในชั้นเรียนของเรา จะเรียนรู้การเขียนโปรแกรมผ่าน jupyter notebook python เป็นหลัก

# อะไรคือ โปรแกรม

โปรแกรมเป็นชุดคำสั่งที่บอกให้เครื่องทำการคำนวนอย่างไร อาจเป็นทั้งทางคณิตศาตร์ หรือเป็นชุดคำนวนเชิงสัญลักษณ์ ของการทำงานบนเครื่องคอมพิวเตอร์ แม้ว่าโปรแกรมอาจถูกเขียนในภาษาที่ต่างกัน แต่มีหลายสิ่งเหมือนกันที่เป็นพื้นฐานสำหรับการเขียนโปรแกรมในทุกๆภาษา

  1. input รับข้อมูลจากคีย์บอร์ด หรืออุปกรณ์ใดๆ

  2. output แสดงผลผ่านทางจอภาพ หรือส่งข้อมูลออกไปสู่อุปกรณ์อื่น

  3. math คณิตศาตร์ที่ทำการดำเนินการข้อมูลหรับปรับเปลี่ยนข้อมูล เช่น บวก ลบ คูณ หาร

  4. conditional execution ตรวจหาเงื่อนไขเฉพาะเพื่อทำงานที่ต้องการ

  5. repetition สามารถทำซ้ำในส่วนต่างๆของโปรแกรม โดยทั่วไปอาจเปลี่ยนแปลงไปตามข้อมูล input
